# **Utils/**
Funciones utilitarias reutilizables en todo el proyecto.

- **validators.js**: Funciones de validación (email, teléfono, CUI, NIT, etc.)
- **helpers.js**: Funciones auxiliares generales
- **responseHandler.js**: Funciones para estandarizar respuestas HTTP (success, error, etc.)
- **logger.js**: Sistema de logging para desarrollo y producción
