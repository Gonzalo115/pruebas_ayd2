# **config/**
Contiene todas las configuraciones del proyecto. 

Aquí van las conexiones a servicios externos, variables de entorno organizadas, y constantes del sistema.

- **database.js**: Configuración y conexión a la base de datos
- **email.js**: Configuración del servicio de correo (SMTP, credenciales)
- **env.js**: Centraliza y valida las variables de entorno
- **constants.js**: Constantes como roles, estados de usuarios, etc.
